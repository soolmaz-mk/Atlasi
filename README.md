# Atlasi
Drawing maps with OpenGL.

## how to run
after installing many libraries

```
g++ Main.cpp -Wfatal-errors -o atlas "common/shader.cpp"  -lopengl32 -lglew32 -lfreeglut -lglu32 -lglfw3 -lgdi32
```
